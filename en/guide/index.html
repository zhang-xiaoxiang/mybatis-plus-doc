# Guide

MyBatis-Plusï¼ˆMP for shortï¼‰is an powerful enhanced tool for MyBatis , Born To Simplify Development.

# Features

  • Non-invasionï¼šNo change but enhancement, no effect for current project by importing MP
  • Little lossï¼šInject CRUD sql automatically when startup, almost non-loss, OOP(Object-Oriented Programming)
  • Powerful CRUDï¼šgeneral Mapper/Service inside, you can CURD by few configuration, powerful condition constructor can simplify programming. for single table CRUD, XML is needless
  • Lambdaï¼šSimplify query by using Lambda, and avoid column name typo
  • Multiple DBï¼šSupport MySQL, MariaDB, Oracle, DB2, H2, HSQL, SQLite, Postgre, SQLServer2005, SQLServer, etc.
  • Auto generate PKï¼š4 strategies for Primary Key, free configuration
  • ActiveRecordï¼šYou can use entity for CRUD, just let entity extends Model
  • Customize general methodsï¼šYou can customize general methods, "Write once, use anywhere"
  • Code Generatorï¼šYou can generate Mapper/Model(Entity)/Service/Controller, and customized code template is also supported(Freemarker, Velocity)
  • Paginationï¼šPhysical pagination, transparent to developers
  • Performance Interceptorï¼šCan show sql execution time (Suggest to open it when doing development)
  • Smart Interceptorï¼šCan stop execution of delete/update for full table, and customized intercepting rule is also supported, which can prevent misoperation
  • Sql Injection Interceptorï¼šCan detect and avoid the sql injection
